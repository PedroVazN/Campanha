# Design System Master File — SGI 4.0

> Direção **Blue Tech** (frontend-design Anthropic).
> Paleta azul tecnológica. **Ganho = verde**. **Perdido = vermelho**.

**Project:** SGI 4.0 — Sistema de Gestão Integrada  
**Style:** Modern tech / financial operations SaaS  
**Stack:** Next.js + Tailwind

---

## Color Palette

| Role | Hex | CSS | Uso |
|------|-----|-----|-----|
| Ink | `#0B1220` | `--ink` | Texto |
| Secondary | `#1E3A5F` | `--secondary` | Hierarquia |
| Accent / Blue | `#2563EB` | `--accent` | CTA, links, UI |
| Accent deep | `#1D4ED8` | `--accent-deep` | Hover CTA |
| Paper | `#E8EEF8` | `--paper` | Fundo |
| Surface | `#FFFFFF` | `--surface` | Painéis |
| Fund / Ganho | `#059669` | `--fund` | Positivo |
| Critical / Perdido | `#DC2626` | `--critical` | Negativo |
| Warn | `#D97706` | `--warn` | Atenção operacional |
| Sidebar | `#071428` | `--sidebar` | Nav |

## Typography

- **Display:** Syne · **UI:** Manrope · **Números:** JetBrains Mono

## Signature

- Filete azul nos títulos e nav ativa (com glow)
- Botão primário em gradiente azul
- Fundo com radiantes azul/ciano
